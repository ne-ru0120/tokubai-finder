# 特売ファインダー 🛒

現在地から **100〜10000m**（入力で変更可）の範囲にある店舗を探し、特売商品を表示する PWA です。
iPhone SE3（375×667pt）向けに最適化しています。

## リンク

- 🌐 **公開サイト（HTTPS）**: <https://ne-ru0120.github.io/tokubai-finder/>
- 📦 **リポジトリ**: <https://github.com/ne-ru0120/tokubai-finder>

> iPhone の Safari で公開サイトを開き、位置情報を許可 → 共有メニュー →「ホーム画面に追加」でアプリとして使えます。

## 構成

| 項目 | 内容 |
|------|------|
| 現在地 | Geolocation API（`watchPosition`で数秒間のベスト測位を採用・`maximumAge:0`）。精度を±Xmで表示し、地図の青ピンをドラッグで手動補正→再検索 |
| 距離計算 | ハバーサイン公式 |
| 全国店舗ディレクトリ | **`data/stores.json`（約28,500店）** を OpenStreetMap から生成（読み取り専用・全国のスーパー＆ドラッグストア） |
| 特売データ | **端末内DB（localStorage）に手入力**。公式チラシを見て登録する |
| 検索結果 | 近い順に **1ページ10店舗・最大10ページ（100店舗）** でページめくり |
| チラシ | 店舗ごとに**トクバイへ直接**遷移（座標を Nominatim で逆ジオ→「チェーン×都道府県×市区」一覧URLを生成し、`?order=near_by_location&latitude&longitude` に**店舗座標**を渡して**その店舗を一覧の先頭**に表示）。トクバイ対応は**約55チェーン**（`TOKUBAI_SLUG`）。未対応チェーンは Google 検索にフォールバック |
| 地図 | Leaflet + OpenStreetMap タイル（無料・キー不要） |
| 店舗の取り込み補助 | OpenStreetMap **Overpass API**（近隣店舗を検索してフォームに反映） |
| 形式 | PWA（ホーム画面に追加可・オフラインで起動可） |

### 全国店舗ディレクトリ（data/stores.json）

全国の **shop=supermarket / chemist / drugstore**（スーパー＆ドラッグストア）約28,500店を
OpenStreetMap(Overpass) から取得しています。生成手順:

1. `fetch_stores.py` … 主要14チェーンを取得（chain/公式URL付き）
2. `fetch_all_stores.py` … 上記に全国の全スーパー＆ドラッグストアを緯度バンド分割でマージ
3. `clean_stores.py` … 国外（韓国・済州島）の混入と駐車場等のノイズを除去

- **OSMに登録済みの店舗のみ**なので全店完全網羅ではありません（大半はカバー）。
- カテゴリは `スーパー` / `ドラッグストア`。チラシリンクは店名から約55チェーンを判定してトクバイ直リンク、それ以外はGoogle検索。
- 検索は「ディレクトリ＋自分で追加した店舗」を統合し、近接重複は自分の店舗を優先します。
- 「✏️ 登録」タブは全件を出さず、**店名検索**または「さがす」結果の「登録する」から各店の特売を入力します。

> **なぜ手入力？** 日本の食品スーパー／ドラッグストアには特売価格を取得できる公開APIが
> 無いため、公式チラシを見て自分で登録する方式にしています（規約クリーン）。初期データとして
> 町田の主要店舗を登録済みです。
> - スーパー: 東急ストア・西友・スーパー三和・Odakyu OX
> - スーパー(追加): 業務スーパー（町田木曽店 約2.4km）・デポー町田（生活クラブ生協 約2.3km）
> - ドラッグストア: マツモトキヨシ・サンドラッグ・スギ薬局・ココカラファイン・クリエイトSD・ウエルシア・トモズ（ミーナ町田店 約100m）・ツルハドラッグ（町田木曽西店 約3.7km）（各公式チラシリンク付き）
>   - 駅から離れた店（ツルハ約3.7km等）は検索距離を広げると表示されます。追加店舗の座標は国土地理院ジオコーディング由来。
>   - 「DEPOT」は生活クラブ生協の食品店「デポー町田」として登録しています（ホームセンター「おうちDEPO」等の別店舗を指す場合は登録画面で修正してください）。
>
> ドラッグストアの追加シードは `ensureSeedBatch()`（`app.js`）で**一度だけ**マージされ、
> ユーザーが編集・削除した店舗は壊しません。`amenity=pharmacy`（調剤薬局）と
> `shop=chemist`（ドラッグストア）はカテゴリで区別され、特売登録はドラッグストア向きです。

## 使い方

- **🔍 さがす**: 距離を選んで「現在地から探す」。範囲内の店舗を近い順に10件ずつページめくり表示し、手入力した特売を見せます。各店舗の「チラシを見る ↗」でその店舗のチラシを開けます。
- **✏️ 登録**: 公式チラシを見て店舗・特売を入力します。
  - 「📍 現在地をセット」で今いる場所の緯度経度を入力欄に反映
  - 「🗺️ 近くから探す」でOSMの近隣店舗一覧から選んで店舗フォームに反映
  - 各店舗カードで商品名・特売価格・通常価格を追加／削除

データはすべて端末内（localStorage）に保存され、外部に送信されません。

### 現在地の精度について

- **PC（デスクトップ）ブラウザはWi‑Fi/IP測位**のため数百m〜の誤差が出ます。**実機のiPhone SE3はGPS**を使うので大幅に正確です。
- アプリは数秒かけて最も精度の高い測位を採用し、結果に **±Xm** を表示します。
- ずれている場合は、**地図の青ピンをドラッグ**すると、その地点から距離を測り直して再検索します。

## ローカルで動かす

Python があればワンコマンドで起動できます。

```powershell
cd E:\Projects\tokubai-finder
python -m http.server 5173
```

ブラウザで <http://localhost:5173> を開きます。
※ 位置情報は `https` か `localhost` でのみ動作します（ブラウザの仕様）。

## iPhone SE3 にインストール（ホーム画面に追加）

1. 同じ Wi-Fi 上の PC で `python -m http.server 5173 --bind 0.0.0.0` を起動
2. iPhone の Safari で `http://<PCのIP>:5173` を開く
   - ※位置情報を使うには **https が必須**。実運用では下記いずれかで公開してください
     - GitHub Pages / Netlify / Cloudflare Pages などに静的ファイルをアップロード
     - `ngrok http 5173` などで https トンネルを張る
3. Safari の共有メニュー →「ホーム画面に追加」

## データの保存先・将来の拡張

- 店舗マスタ: `localStorage["tf_stores_v1"]`（`{id,name,category,lat,lon,url}` の配列）
- 特売: `localStorage["tf_sales_v1"]`（`{ 店舗id: [{id,name,price,was}] }`）

将来、複数端末で共有したくなったら、`loadStores()/saveStores()/loadSales()/saveSales()`
（`app.js`）を自前のサーバーAPI呼び出しに差し替えるだけで対応できます。
